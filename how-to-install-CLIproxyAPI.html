⚡ CLIProxyAPI — Cài đặt nhanh

Ubuntu • systemd 24/7 • UFW • Web UI

1

Cập nhật hệ thống

sudo apt update && sudo apt upgrade -y
sudo apt install -y wget curl tar unzip
2

Tạo thư mục dự án & cd vào

mkdir -p ~/cliproxyapi
cd ~/cliproxyapi
3

Tải CLIProxyAPI mới nhất

wget https://github.com/router-for-me/CLIProxyAPI/releases/download/v7.2.94/CLIProxyAPI_7.2.94_linux_amd64.tar.gz
4

Giải nén & kiểm tra

tar -xzf CLIProxyAPI_7.2.94_linux_amd64.tar.gz
chmod +x cli-proxy-api
ls -lh
✅ Phải thấy file cli-proxy-api trong danh sách.
5

Mở port UFW

sudo ufw allow 8317/tcp
sudo ufw status
6

Tạo file config.yaml

nano ~/cliproxyapi/config.yaml

Dán nội dung sau:

host: ""
port: 8317
auth-dir: ~/cliproxyapi
api-keys:
  - your-secret-api-key
debug: false
logging-to-file: true
routing:
  strategy: round-robin
remote-management:
  allow-remote: true
  secret-key: "your-management-key"
  disable-control-panel: false
📝 Ctrl+O → Enter → Ctrl+X
7

Chạy thử dịch vụ

cd ~/cliproxyapi && ./cli-proxy-api -config config.yaml
✅ Thấy Listening on :8317 là OK! Giữ nguyên terminal này.

Mở terminal thứ hai, kiểm tra health:

curl http://127.0.0.1:8317/health
✅ Kết quả: {"status":"ok"}

Xong thì về terminal cũ nhấn Ctrl+C để dừng.

8

Tạo file systemd service

sudo nano /etc/systemd/system/cliproxyapi.service

Dán nội dung sau:

[Unit]
Description=CLIProxyAPI
After=network-online.target
Wants=network-online.target

[Service]
Type=simple
User=root
Group=root
WorkingDirectory=/root/cliproxyapi
ExecStart=/root/cliproxyapi/cli-proxy-api -config /root/cliproxyapi/config.yaml
Restart=always
RestartSec=5
LimitNOFILE=65535
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
📝 Ctrl+O → Enter → Ctrl+X
9

Kích hoạt service 24/7

sudo systemctl daemon-reload
sudo systemctl enable cliproxyapi
sudo systemctl start cliproxyapi
sudo systemctl status cliproxyapi
✅ Phải thấy active (running) — từ giờ chạy 24/7!

Xem log nếu muốn:

journalctl -u cliproxyapi -f
# Ctrl+C để thoát xem log
10

Kiểm tra IP binding

ss -tulpn | grep 8317
✅ Phải thấy 0.0.0.0:8317 — truy cập được từ máy khác.
⚠️ Nếu chỉ thấy 127.0.0.1: sửa host: "" trong config.yaml, rồi sudo systemctl restart cliproxyapi.
11

Đăng nhập Provider & Management Key

Tạo Management Key (để đăng nhập Web UI):

echo "mk-$(openssl rand -hex 24)"

Copy kết quả, chạy:

nano ~/cliproxyapi/config.yaml
# Sửa dòng secret-key: "mk-3d0a9b6..." (chuỗi vừa tạo)

Sau đó restart service:

sudo systemctl restart cliproxyapi

Đăng nhập Provider:

cd ~/cliproxyapi

# Claude
./cli-proxy-api -claude-login

# hoặc Gemini
./cli-proxy-api -gemini-login

# hoặc Codex
./cli-proxy-api -codex-login
⚠️ Cần có tài khoản / API key của provider tương ứng.
12

Truy cập Web UI

Giao diệnĐịa chỉ
Trang chủhttp://<IP>:8317
Web UI quản trịhttp://<IP>:8317/management.html

Ví dụ: http://192.168.1.20:8317/management.html

🔑 Đăng nhập bằng Management Key đã tạo.
🔌

Các API Endpoint

Base URL: http://<IP>:8317

Phương thứcEndpointMô tả
POST/v1/chat/completionsChat Completions (OpenAI compatible)
GET/v1/modelsDanh sách model
GET/healthHealth check
GET/management.htmlWeb UI quản trị
curl http://127.0.0.1:8317/v1/models \
  -H "Authorization: Bearer your-secret-api-key"

sudo reboot
