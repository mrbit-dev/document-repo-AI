Cài đặt OmniRoute trên Ubuntu Server (24/7)

1. Cập nhật hệ thống

sudo apt update
sudo apt upgrade -y

2. Cài Node.js 22 LTS (kèm npm)

curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs

node -v
npm -v

Lưu ý: npm sẽ được cài cùng Node.js.

3. Cài OmniRoute

sudo npm install -g omniroute

omniroute --version

4. Chạy thử

omniroute

5. Chạy OmniRoute 24/7 bằng systemd

Tạo file service:

sudo nano /etc/systemd/system/omniroute.service

Nội dung:

[Unit]
Description=OmniRoute AI Gateway
Documentation=https://github.com/diegosouzapw/OmniRoute
After=network-online.target
Wants=network-online.target

[Service]
Type=simple

# User chạy service
User=root
Group=root

# Thư mục làm việc
WorkingDirectory=/root

# Đọc file .env nếu có
EnvironmentFile=-/root/.omniroute/.env

# Một số biến môi trường
Environment=NODE_ENV=production
Environment=NO_COLOR=1

# Chạy OmniRoute
ExecStart=/usr/bin/omniroute --no-open

# Tự restart khi lỗi
Restart=always
RestartSec=5

# Timeout khi dừng
TimeoutStopSec=20

# Log
StandardOutput=journal
StandardError=journal

# Tăng giới hạn file
LimitNOFILE=65535

[Install]
WantedBy=multi-user.target

6. Kích hoạt service

sudo systemctl daemon-reload
sudo systemctl enable omniroute
sudo systemctl start omniroute

7. Kiểm tra trạng thái

systemctl status omniroute

8. Xem log realtime

journalctl -u omniroute -f

9. Quản lý service

sudo systemctl restart omniroute
sudo systemctl stop omniroute
sudo systemctl disable omniroute

Nếu gặp lỗi Unexpected token 'with', hãy kiểm tra đang dùng Node.js 22 trở lên và cài lại OmniRoute.
