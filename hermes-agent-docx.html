Hermes Agent Deployment Guide

Hướng dẫn triển khai Hermes Agent trên VPS Ubuntu dành cho người mới và quản trị viên hệ thống.

1. Tổng quan về Hermes Agent

Hermes Agent là AI Agent Framework của Nous Research, hỗ trợ quản lý Agent, MCP Server, Tool Calling, Session và Dashboard Web trực quan.

  • Dashboard Web
  • MCP Integration
  • Tool Execution
  • Session Management
  • Cloud & Local LLM

4. Cài đặt vào VPS

Bước 1: Tạo User

sudo adduser luna
sudo usermod -aG sudo luna
su - luna

Bước 2: Cài đặt Hermes Agent

curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash

uv tool install hermes-agent

hermes --help

Khởi động Dashboard

hermes dashboard

5. Truy cập Dashboard từ PC bằng SSH Tunnel

Sau khi Dashboard chạy trên VPS:

ssh -L 9119:127.0.0.1:9119 luna@165.99.16.113

Mở trình duyệt:

http://localhost:9119
Không cần expose Dashboard trực tiếp ra Internet khi sử dụng SSH Tunnel.

6. Mở Port 9119

sudo ufw allow 9119/tcp
sudo ufw reload

Kiểm tra:

ss -tulpn | grep 9119

7. Troubleshooting

Kiểm tra Dashboard

hermes dashboard --status

Kiểm tra Port

sudo lsof -i :9119

Kiểm tra từ VPS

curl http://127.0.0.1:9119

Lưu ý bảo mật

  • Không dùng --insecure trên VPS public.
  • Nên sử dụng SSH Tunnel.
  • Không chia sẻ API Key.
